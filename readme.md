# Devcamper API

> Backend API for the Devcamper Application, which is a Bootcamp directory website

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings of your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

#Run in prod mode
npm start
```
